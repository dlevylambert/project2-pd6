Hello! This app has been made by Win Wang, Amanpreet Kandola, Daelin Fischman, and Kevin Huang. We are Team Awesome, and welcome to our app!

This miniature search engine takes a string input, and searches GiantBomb, a gaming website, for games relating to the string. For example, inputting "Halo" will return games such as "Halo 4" and "Halo: Reach".
On the results page, clicking the name of the game will redirect the user to a google search with the name as the parameter. Clicking "Description" will redirect the user to the game's description that GiantBomb's API gives the app.
